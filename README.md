# Bicholândia 🐾

A **Bicholândia** é uma plataforma web institucional desenvolvida para uma ONG fictícia de proteção animal, fundada em 2026 na região de Cornélio Procópio - PR. O projeto visa facilitar a adoção responsável de animais resgatados, além de centralizar informações sobre doações e cadastro de voluntários.

---

## 🎨 Identidade Visual e Design

O projeto apresenta uma interface moderna com foco na experiência do usuário (UX), utilizando cores que transmitem tranquilidade e botões de destaque para ações principais.

### 🌈 Paleta de Cores
| Cor | Hex | Aplicação |
| :--- | :--- | :--- |
| **Petróleo Escuro** | `#0E3032` | Header, Footer e Cards |
| **Azul Oceano** | `#4994A9` | Background das seções principais |
| **Destaque Amarelo** | `#FFAA00` | Botões e Links importantes (Hover) |

### ✒️ Tipografia
- **Títulos:** `League Spartan` e `Roboto` (via Google Fonts).
- **Corpo:** `Arial / Helvetica`.

---

## 🛠️ Tecnologias Utilizadas

O desenvolvimento foi focado em técnicas puras de Front-end para garantir performance e total controle sobre o estilo:

*   **HTML5:** Estrutura semântica e acessível.
*   **CSS3 Avançado:**
    *   **CSS Grid:** Utilizado na vitrine de animais para um layout responsivo automático.
    *   **Flexbox:** Alinhamento de formulários de login/cadastro e menus.
    *   **Microinterações:** Efeitos de transição (*hover*) dinâmicos nos botões e links.
    *   **Responsividade:** Uso de `auto-fit` e `minmax` para adaptação em diferentes telas.
